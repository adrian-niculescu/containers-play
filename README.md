## Setup

Starting from a machine running Ubuntu (tested on Ubuntu 20.04.3 LTS), run the following to install and configure [Docker](https://www.docker.com/), [Microk8s](https://microk8s.io/).

```bash
bash setupHostMachine.sh
```
